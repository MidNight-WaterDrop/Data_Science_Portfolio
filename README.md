## Local Setup
1. Clone: `git clone https://github.com/yourusername/yourrepo.git`
2. Navigate: `cd yourrepo`
3. Install: `bundle install`
4. Run: `jekyll serve`
5. Visit: http://localhost:4000  
   Requires Jekyll ([install guide](https://jekyllrb.com/docs/installation/)).
